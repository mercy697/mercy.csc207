type of lists

ordered list
Used for information for which order is important.
Create an Ordered List using <ol></ol>
Each item of the list should be specified with the list item command <li>
Ordered list uses Arabic numerals as its default numbering type. Two main attributes for OL are type and start Attribute values for type are 1, A, a, I, or i
The start attribute can be used to continue a previous numbering.
The value of each item in the list can also be changed by using the value attribute in the <li>tag.
Ordered lists can also be nested in which case, each sub list can use a different type of numbering using the type attribute
syntax: <ol>
<li> item 1</li>
<li>item 2</li>
<li>item 3</li>
</ol>
example of ordered list is shopping list
  1. phone
  2. bag
  3. foodstuffs
  4. stationaries
unordered list
Used for information when each item is equally important.
Each item of the list should be specified with the list item command <li>.
Main attribute for UL is type with values:
disc, circle or square
the default bullet type is disc
example of unordered list is
  • Apple
  • Orange
  • Grapefruit
definition list
Displays a number of terms and associated definition
. Renders without bullets.
Create definition lists using <dl></dl>
The <dt></dt> tag creates a definition term in a definition list;
The <dd></dd> tag defines the definition for the term and will appear indented and below the term.
Syntax: <dl>
<dt>term 1</dt>
<dd>explanation …</dd>
<dt>term 2</dt>
<dd>explanation …</dd>
</dl>
